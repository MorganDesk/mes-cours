# Calculer la valeur d'une expression littérale
Pour calculer la valeur d’une expression littérale pour une valeur donnée, on remplace la lettre par le nombre en remettant les signes de multiplication $\times$ sous-entendus.

>[!exemple] Exemple :
>Calculer $A=5(7-2x)+9$ pour $x=3$ :
>$$\begin{aligned}A&=5 \times (7-2\times 3)+9\\&=5 \times 1 +9\\&=5+9\\&=14\end{aligned}$$
